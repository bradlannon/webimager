## Integration Issues (3 found)

### Issue 1: applyResize breaks background removal state
- **Location:** `src/store/editorStore.ts` applyResize
- **Impact:** BGREM-01, BGREM-06, EXPT-01, EXPT-02
- **Problem:** Resize bakes a new bitmap without mask compositing, leaves `backgroundRemoved: true` with stale mask
- **Fix:** Clear background removal state in applyResize's `set()` call after creating new bitmap

### Issue 2: restoreBackground uses toggle instead of clear
- **Location:** `src/hooks/useBackgroundRemoval.ts` line 145
- **Impact:** BGREM-05
- **Problem:** `toggleBackground()` is bidirectional — double-click re-applies removal silently
- **Fix:** Use `clearBackgroundMask()` instead of `toggleBackground()`

### Issue 3: Dead drawCheckerboard import
- **Location:** `src/hooks/useRenderPipeline.ts` line 3
- **Impact:** Tech debt (BGREM-04 works via CSS path)
- **Fix:** Remove unused import
